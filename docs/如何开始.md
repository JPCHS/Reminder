# 如何开始

1. 前往该项目的[GitHub界面](https://github.com/jpchs/Reminder)的Release中下载最新发布版本，选择`Reminder_setup.exe`

2. 打开文件，按提示进行安装

   ![setup_1](./setup_1.png)

   ![setup_2](./setup_2.png)

3. 完成安装

   ![setup_3](./setup_3.png)

   ![setup_4](./setup_4.png)

4. 重启或直接执行
